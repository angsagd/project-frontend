A. Deskripsi

Website ini berfungsi sebagai pusat informasi untuk satu game tertentu. Situs ini mencakup berita terbaru, daftar karakter/unit, panduan bermain, turnamen, dan komunitas.

B. Halaman (Statis)

  1. Home – Ringkasan game, berita terbaru, dan update patch.
  2. Karakter/Unit – Daftar karakter/unit dalam game beserta peran dan statistiknya.
  3. Panduan & Build – Tips strategi bermain, build item, dan kombinasi skill terbaik.
  4. Turnamen & Event – Informasi turnamen, jadwal kompetisi, dan hadiah.
  5. Komunitas – Forum diskusi, daftar guild/clan, dan tim eSports.

C. Fitur (Interaktivitas JavaScript)

  • Slideshow Berita Game di Halaman Home – Gambar dan judul berita akan berganti otomatis/manual.
  • Hover Animasi pada Daftar Karakter/Unit – Saat kursor diarahkan ke karakter/unit, akan muncul efek highlight dan deskripsi singkat.
  • Filter Karakter Berdasarkan Role – Pengguna dapat menyaring daftar karakter (Tank, DPS, Support).
  • Accordion/Dropdown untuk Panduan & Build – Panduan strategi bisa dibuka dan ditutup sesuai kategori.
  • Countdown Timer ke Turnamen Berikutnya – Menampilkan hitung mundur hingga turnamen dimulai.

D. Detail Implementasi

1. Home

  • Header dengan gambar banner game dan update terbaru.
  • Slideshow berita dan patch terbaru.
  • Tombol "Mulai Bermain" atau "Pelajari Game" untuk mengarahkan ke halaman Panduan & Build.

2. Karakter/Unit (contoh)

  • Grid layout untuk daftar karakter/unit berdasarkan role.
  • Filter karakter berdasarkan kategori (Tank, DPS, Support).
  • Hover effect untuk menampilkan informasi singkat karakter.

3. Panduan & Build

  • Accordion atau dropdown untuk menampilkan strategi berdasarkan peran atau mode permainan.
  • Tombol "Copy Build" untuk menyalin build item terbaik ke clipboard.

4. Turnamen & Event

  • Tabel interaktif berisi informasi tanggal, lokasi, dan hadiah turnamen.
  • Countdown timer yang menunjukkan sisa waktu sebelum turnamen dimulai.

5. Komunitas

  • Daftar forum diskusi dan guild dalam format grid.
  • Tooltip interaktif saat mengarahkan kursor ke nama tim/guild.

E. Syarat yang Harus Dipenuhi

  • Minimal memiliki 5 halaman HTML yang saling terhubung.
  • Menggunakan CSS Grid atau Flexbox untuk tampilan yang rapi dan responsif.
  • Navigasi antar halaman harus jelas dan mudah digunakan.
  • Filter karakter dan efek expanding panduan harus berfungsi dengan baik.
  • Minimal ada 3 fitur interaktif yang berjalan dengan baik (slideshow berita, filter karakter, countdown turnamen, dll.).
